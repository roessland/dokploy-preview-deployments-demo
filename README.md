# dokploy-preview-deployments-demo

Simple Python Flask app for testing Dokploy preview deployments.

## Setup

Install build tools:

```bash
# Install nixpacks
brew install nixpacks

# Install railpack
curl -sSL https://railpack.com/install.sh | sh
```

## Commands

```bash
# List available commands
just

# Build with nixpacks
just nixpack
```
